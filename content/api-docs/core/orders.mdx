## Orders

An order represents a payment between two or more parties. Use the Orders API to create, update, retrieve, authorize, and capture orders.

## Authentication

You need an API key. You can obtain your API by signing up for an account on our platform. Use  This API key in the Authorization header of your HTTP requests.

```shell 

Authorization: Bearer YOUR_API_KEY

```
