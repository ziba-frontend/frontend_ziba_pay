## Transaction Search


Use the Transaction Search API to get the history of transactions for a PayPal account. To use the API on behalf of third parties, you must be part of the PayPal partner network. Reach out to your partner manager for the next steps. To enroll in the partner program, see Partner with PayPal. For more information about the API, see the Transaction Search API Integration Guide.

## QUERY PARAMETERS

<div className="grid grid-cols-2  gap-10 ">
<p>transaction_id</p>

<div ><p>string  17 .. 19  characters
Filters the transactions in the response by a PayPal transaction ID. A valid transaction ID is 17 characters long, except for an order ID, which is 19 characters long.</p> <div className="bg-br p-1 rounded"><p><span className="text-2xl">Note : </span>A transaction ID is not unique in the reporting system. The response can list two transactions with the same ID. One transaction can be balance affecting while the other is non-balance affecting.</p></div></div>
<p>transaction_type</p>
<div ><p>string
Filters the transactions in the response by a PayPal transaction event code. See Transaction event codes.</p></div>
</div>
</div>
