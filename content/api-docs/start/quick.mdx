This quick start guide will help you integrate Ziba Pay into your application with minimal setup. Follow the steps below to get started.

## Step 1: Sign Up for Ziba Pay

First, sign up for a Ziba Pay account at the [Ziba Pay website](https://www.zibapay.com). Once you have an account, navigate to the API section to obtain your API keys.
