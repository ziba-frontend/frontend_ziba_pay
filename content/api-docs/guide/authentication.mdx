## Basic Authetication

Basic authentication involves sending a username and password with each request. While this method is straightforward.
To authenticate using Basic Authentication, include your username and password in the request headers.

Example

```shell
curl -X GET https://api.yourdomain.com/endpoint \
  -H "Authorization: Basic $(echo -n 'username:password' | base64)"
  ```

Nodejs example

```js
const axios = require('axios');

const username = 'your_username';
const password = 'your_password';

const response = await axios.get('https://api.yourdomain.com/endpoint', {
  headers: {
    Authorization: `Basic ${Buffer.from(`${username}:${password}`).toString('base64')}`,
  },
});

console.log(response.data);
```
